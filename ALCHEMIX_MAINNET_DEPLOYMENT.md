## Key Functions

### Deposit Functions
- `depositUnderlying(yieldToken, amount, recipient, minimumAmountOut)` - Deposit collateral
- `deposit(yieldToken, amount, recipient)` - Deposit yield tokens directly

### Borrow Functions
- `mint(amount, recipient)` - Borrow alUSD/alETH (called "mint" in Alchemix)
- `mintFrom(owner, amount, recipient)` - Borrow on behalf of another account

### Repay Functions
- `repay(token, amount, recipient)` - Repay debt with underlying
- `burn(amount, recipient)` - Burn alUSD/alETH to repay

### Query Functions
- `accounts(owner)` - Get account details (debt, totalDeposited)
- `positions(owner, yieldToken)` - Get position for specific collateral
- `totalValue(owner)` - Get total collateral value
- `convertSharesToUnderlyingTokens(yieldToken, shares)` - Convert shares to tokens

### Health & Limits
- `getCdpTotalDebt(owner)` - Get total debt
- `getCdpTotalCredit(owner)` - Get borrowing capacity
- `minimumCollateralization()` - Get minimum collateralization ratio (200% = 2e18)

## Testing Strategy

```bash

anvil --fork-url https://eth.llamarpc.com --fork-block-number 18000000
```

### For Farcaster Integration
1. Deploy Frame backend to Vercel
2. Use mainnet contracts with **small test amounts** (0.01 ETH, $10 USDC)
3. Test deposit → borrow → repay flow
4. Monitor transactions on Etherscan

## Important Notes

⚠️ **No Testnet Deployment**: Alchemix V2 is only on mainnet. For testing:
- Use mainnet fork with Foundry/Anvil
- Use small amounts on mainnet ($10-50)
- Test thoroughly on fork before mainnet

⚠️ **Collateralization**: Alchemix requires 200% collateralization (can borrow 50% of collateral value)

⚠️ **Self-Repaying**: Loans automatically repay from yield generated by deposited collateral
